# OPTIONAL MANIFEST FIELDS FOR COPYRIGHT SUPPORT (AII Standard)

These fields may be added to `manifest.json` for AII:

"human_authorship_statement": "",
"human_signature": "",
"creator_email": "",
"editorial_notes": "",
"human_editing_steps": [],
"human_curated_output": true

These optional fields help creators document human involvement in AI-assisted image creation.
